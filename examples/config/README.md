# Example configurations

Copy any of these into `~/.pier/config.toml` and adjust to taste. See [docs/configuration.md](../../docs/configuration.md) for the full list of options.

| File | Use it when… |
| --- | --- |
| `minimal.toml` | You just want to pick a model and accept the safe defaults. |
| `cautious.toml` | You want the agent to ask before doing anything that writes or runs. |
| `power-user.toml` | You trust the workspace sandbox and want fewer prompts. |

These are starting points, not the only valid settings — mix and match.

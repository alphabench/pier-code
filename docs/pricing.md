# Pricing

For full, current pricing see [www.piercode.com/pricing](https://www.piercode.com/pricing).

## Pro — $5 / month

Everything you need to run Pier as your daily coding agent:

- Powered by sovereign Indian models
- Multi-repo & monorepo context
- Indian-language prompting
- Email support

Pro includes a monthly pool of usage credits that covers typical day-to-day coding.

## Max — $20 / month

For heavy, all-day use — everything in Pro, with a much larger monthly pool of usage credits so you rarely need to top up.

- Everything in Pro
- ~4× the monthly usage credits
- Built for full-time, high-volume coding

## Usage pack — +$5, any time

Out of monthly credits? Add more usage whenever you need it:

- Stack as many packs as you need
- Same per-token rates as Pro
- Credits never expire
- Buy mid-session, no interruption

## Why it's affordable

Sovereign Indian models are trained and hosted in Bharat and priced far below frontier US models. A coding agent reads far more than it writes, so low input-token prices compound across every repo read, diff, and tool result.

For the hardest long-horizon tasks, you can plug in a frontier model per run — so you only pay frontier prices when you actually need frontier quality.

# Prompting in your language

**AI for all, from Bharat.** Pier is built for how India codes. You can describe tasks in the language you think in — the agent reasons over your intent regardless of the language you type in.

## Supported languages

You can prompt in:

- English
- Hindi (हिन्दी)
- Tamil (தமிழ்)
- Kannada (ಕನ್ನಡ)
- Telugu (తెలుగు)
- …and more

You can also mix languages freely within a prompt — for example, English technical terms inside a Hindi sentence.

## Examples

English:

> Add input validation to the signup form and write a test for it.

Hindi:

> साइनअप फ़ॉर्म में इनपुट वैलिडेशन जोड़ो और उसके लिए एक टेस्ट लिखो।

Tamil:

> signup form-க்கு input validation சேர்த்து, அதற்கு ஒரு test எழுது.

The agent's code, identifiers, and comments follow your project's existing conventions — prompting in an Indian language changes how *you* communicate intent, not the style of the code it writes (unless you ask for that too).

## Notes

- Diffs, command output, and tool results are shown as-is.
- If you want responses or explanations in a specific language, just ask.
